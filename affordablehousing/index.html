Visualizing Need for Housing In Pennslyvania / An interactive data index

When considering affordable housing in Pennsylvania, we asked ourselves: how does need compare to supply across the state of PA, and how has need for affordable housing changed over time? Can we make an argument about the spatial distribution of need in PA that will be useful to state and national policymakers when targeting new programs to underserved populations?

Using PUMs data and ACS data, we will make a chloropleth map of Pennsylvania that changes over time to reflect changing need. Symbolized on the map will be data that starts to uncover where the people with the greatest level of housing insecurity are living in the state, which will be toggled on and off to see layers separately, or join them together into an index so that they can be viewed simulataneously. A slider timeline and toggle buttons will allow the viewer to interact with the data and learn more about where people need housing in Pennsylvania. Clicking on an individual county shape will uncover more information about how the % change in renter/housing population has changed compared to the % change in rental/housing units.
